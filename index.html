svgLoader.js

A simple JavaScript file to create a properly centered loader image.

AJAX loading spinners should be easy

Generally, however its more work than its worth often if you're using GIFs, you have to wait for the images to load and you can't use alpha transparency. Animated PNGs are a bust so you're left with using JavaScript. There are many good canvas scripts our there but this one uses SVG. Neither SVG nor canvas are supported in IE8 but you have many options to work around.

This library creates a loader similar to what you find in Android ICS devices, two counter rotating arced gradients. With this library, you can create the loader of any size and color you wish and it will be vertically and horizontally centered in the element in which you place it. Ideally the loading container should contain nothing else, otherwise you may need to add some css of the form:

#Loader svg{
  position:absolute;
  top: 0px;
  left: 0px;
  z-index: 0
}

You can use the loader like this:

<div id='Example1'></div>
<script>
    svgLoader(
        document.getElementById('Example1'),    //Container for loader
        100,                                    //height of spinner in px
        100,                                    //width of spinner in px
        100,                                    //number of sections to divide the arcs into
        360,                                    //the number of degrees the arc will cover
        70,                                     //the inner radius of the arc
        90,                                     //the outer radius of the arc
        '#000000',                              //the color of the arc
        0.5,                                    //the opacity of the leading section 
        '4s',                                   //the time it takes for the clock-wise arc to complete a circle
        '2s'                                    //the time it takes for the counterclock-wise arc to complete a circle
    );
</script>
